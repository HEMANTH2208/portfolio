# Project Images

This directory contains images for portfolio projects.

## Structure
- Each project should have its own subdirectory or use a naming convention like `project-name-1.jpg`
- Supported formats: JPG, PNG, WebP, GIF
- Recommended dimensions: 1200x800px for main images, 400x300px for thumbnails

## Current Projects
- `outpass-pro-*.jpg` - OutPass Pro project screenshots
- `linkedin-automation-*.jpg` - LinkedIn Automation project screenshots

## Adding New Images
1. Upload your project images to this directory
2. Update the project data in `assets/data/projects.json`
3. Use the admin panel to manage project images

## Image Optimization
- Compress images for web use
- Use WebP format when possible for better performance
- Provide alt text for accessibility
