# Certificate Images

This directory contains certificate files and images.

## Structure
- Use descriptive filenames like `certificate-name-issuer.pdf`
- Supported formats: PDF, JPG, PNG
- Keep original quality for certificates

## Current Certificates
- `hackfinity-2025.pdf` - HACKFINITY 2025 certificate
- `design-thinking-nptel.pdf` - NPTEL Design Thinking certificate
- `microsoft-productivity-nsdc.pdf` - Microsoft Digital Productivity certificate
- `ieee-xtreme.pdf` - IEEEXtreme Programming certificate

## Adding New Certificates
1. Upload certificate files to this directory
2. Update the certificate data in `assets/data/certificates.json`
3. Use the admin panel to manage certificates

## File Guidelines
- PDF format preferred for official certificates
- High-resolution images for scanned certificates
- Secure storage for sensitive documents
