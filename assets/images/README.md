## ğŸ¨ Image Guidelines

### **Formats**
- **Icons:** PNG with transparency
- **Photos:** JPG for photographs, PNG for graphics
- **Favicons:** ICO format for main favicon

### **Optimization**
- Compress images for web delivery
- Use appropriate dimensions
- Maintain aspect ratios
- Consider retina displays (2x resolution)

### **Naming Convention**
- Use lowercase with hyphens
- Descriptive names
- Include dimensions when relevant
- Example: `icon-192x192.png`
